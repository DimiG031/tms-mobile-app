# Business dokumentacija mobilne aplikacije

## Pregled

Ovaj folder opisuje poslovne module mobilne aplikacije za TMS sistem.

Dokumentacija je pisana iz ugla korisnika mobilne aplikacije, pre svega vozača, ali uz podršku za budući role-based interfejs za druge korisnike.

## Pravila dokumentacije

* koristiti srpsku latinicu
* ne prikazivati interne ID vrednosti kao korisnički koncept
* razlikovati operativna obaveštenja od chat poruka
* jasno navesti kada je potreban backend endpoint
* dokumentovati ponašanje koje je korisniku važno na telefonu
